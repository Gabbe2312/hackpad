# Keymap Reference

## General Controls

| Key | Function |
|------|----------|
| M | Switch between IP and Web tool banks |
| P | Toggle between IP and Port edit mode |
| E | Save current input |
| B | Backspace |
| C | Clear current input |
| h | Execute selected tools |

## Tool Selection Keys

| Key | IP Bank | Web Bank |
|------|---------|----------|
| n | Nmap | sqlmap |
| i | Nikto | WhatWeb |
| g | Gobuster | theHarvester |

## Input

- Numeric keys (0–9) → IP / Port input
- '.' → IPv4 separator
